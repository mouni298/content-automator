# Background music

Drop royalty-free `.mp3` / `.m4a` tracks here. The assembler picks one and mixes
it under the voiceover at low volume (see `config.yaml: video.music_volume`).

Use tracks cleared for monetized use to avoid YouTube Content ID claims, e.g.:
- YouTube Audio Library (free, license-cleared)
- Pixabay Music, Incompetech (Kevin MacLeod, attribution)

Cinematic / ambient / orchestral tracks fit the history-documentary tone best.
